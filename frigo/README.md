# Le frigo

* Le matériel :
	* 10 plaques de polystirène de 120*60 cm et 27 mm d'épaisseur
	* Planches de bois pour le fond (120*95)
	* Tasseaux de bois de 22*22 mm
	* Colle à bois
	* Colle à polystirène

* Dimensions finales : 120*95*60
