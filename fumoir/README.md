# Schermiam-miam
## Fumoir
### Le matériel :

* Planches de coffrage (non traitées):
	* ici 12 planche 27*200 sur 4m de long

* Tasseaux de 65*65 pour les angles
* Tasseaux plus fin pour fixer les planches entre-elles

* Dimensions :
	* 800*800 à l'intérieur sur 2m de haut
